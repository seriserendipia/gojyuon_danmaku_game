# 弹幕互动游戏：日语五十音记忆训练

通过网课学日语五十音，记忆效果有限。提问抢答需要老师手动播放音频、记分，会减慢课堂效率。

尝试用b站直播弹幕互动游戏的形式，让课堂里的同学们都参与进来

- b站弹幕获取使用了 blivedm   
- GUI 用 PyQt5 写的  
- 五十音的音频来自 https://www.coscom.co.jp/hiragana-katakana/au50on/
